# WRCF Industry Default Selection

## Overview
<!-- One paragraph: what this enhancement does, which existing feature it improves, and why. -->
a. Industry WRCF Dashboard we have selection of Industry Sector and Industry when user click edit structure form quick action 
need to send the default selected value to the Manage Industry WRCF and need to load the data as per the selected values.

b.Industry WRCF Dashboard we have selection of Industry Sector and Industry when user click Manage Roles form quick action 
need to send the default selected value to theManage WRCF Roles and need to load the data as per the selected values.


## Current Behaviour
<!-- Describe how the feature works today before this enhancement. -->
When user open Manage Industry WRCF ad Manage WRCF Roles form quick action on Industry WRCF Dashboard screen's filter,
it is not getting the select value.

## Desired Behaviour
<!-- Describe how it should work after the enhancement. Be specific. -->
a. Manage Industry WRCF.
- When user select the filter of Industry WRCF Dashboard and click on the edit structure form quick action
the selected value should be transfer to the Manage Industry WRCF ad it filter should be applied as per the selected value.
- All the column data should be loaded as per that selected value. 
- Use routing to share the data form the one component to another component.
- In Manage Industry WRCF screen all column data should be selected for the first value and load next columns data as per the last selected value.

b. Manage WRCF Roles.
- When user select the filter of Industry WRCF Dashboard and click on the edit structure form quick action
the selected value should be transfer to the Manage Industry WRCF ad it filter should be applied as per the selected value.
- All the column data should be loaded as per that selected value. 
- Use routing to share the data form the one component to another component.
- In Manage WRCF Roles screen all column data should be selected for the first value and load next columns data as per the last selected value.

## Default Behaviours
<!-- Auto-select rules, pre-population, initial state changes if any. -->
a. Both Manage Industry WRCF / Manage WRCF Roles loads as per the selected value form Dashboard the column's data should load.
b. Each columns data should load data for previous column selected data.
